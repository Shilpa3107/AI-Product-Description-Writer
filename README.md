# AI Product Description Writer

Welcome to the AI Product Description Writer! This application is a powerful tool designed to help e-commerce professionals, marketers, and copywriters generate, optimize, and test product descriptions using the power of generative AI.

## Key Features

-   **Generate Descriptions**: Create compelling and engaging product descriptions from scratch by providing basic product details like name, features, and target audience.
-   **Optimize for SEO**: Refine your existing descriptions to improve search engine visibility by incorporating target keywords.
-   **A/B Test Variants**: Generate multiple versions of a product description to test and identify the highest-converting copy.
-   **Manage Templates**: Save and manage your favorite description templates for future use.

## Getting Started

To get started with the application, you'll need to install the dependencies and run the development server.

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```

The application will be available at [http://localhost:9002](http://localhost:9002).
